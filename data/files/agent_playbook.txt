== OBJECTION HANDLING ==
Objection: "We already have a data warehouse"
Response: MCP Data Connector sits on TOP of your existing warehouse —
          it doesn't replace it. We add the agent access layer.

Objection: "Security concerns about AI accessing our data"
Response: Every tool call is logged, permissioned, and auditable.
          Agents only see what you explicitly allow.

Objection: "Too complex to integrate"
Response: Average integration time is 2 hours. We provide connectors
          for Postgres, REST APIs, and file systems out of the box.
